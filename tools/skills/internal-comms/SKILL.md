# Internal Communications

Help write all kinds of internal communications using professional formats.

## Supported Types

- **3P Updates**: Progress, Plans, Problems
- **Status Reports**: Regular team/project updates
- **Leadership Updates**: Executive summaries
- **Company Newsletters**: Organization-wide comms
- **FAQs**: Common questions and answers
- **Incident Reports**: Post-incident documentation

## Process

1. Identify the communication type
2. Gather necessary information
3. Follow the appropriate format
4. Keep it concise and actionable

## Best Practices

- Lead with the most important information
- Use bullet points for scannability
- Include clear next steps or asks
- Keep paragraphs short
- Use headers to organize content
