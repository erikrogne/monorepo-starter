# Brainstorming

Turn ideas into designs through collaborative dialogue.

## When to Use

Before any creative work:
- Creating new features
- Building components
- Adding functionality
- Modifying behavior

## Process

### 1. Understand Intent

Ask clarifying questions:
- What problem are you solving?
- Who is the user?
- What does success look like?

### 2. Explore Options

Generate multiple approaches:
- List 3-5 different solutions
- Include pros/cons for each
- Consider constraints

### 3. Refine Together

- User selects preferred direction
- Iterate on details
- Document decisions

## Best Practices

- Start broad, then narrow
- No bad ideas in brainstorming
- Quantity before quality
- Build on each other's ideas
